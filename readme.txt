Bank Statement Web App

This application is used to clean bank statement Excel files and
generate party-wise summaries.

Key functions:
- Removes bank codes and transaction identifiers
- Normalises party names
- Aggregates transactions party-wise
- Produces audit- and GST-ready outputs

Usage:
Upload the Excel bank statement into the web application
Review the cleaned data
Download the results

This tool follows firm-standard accounting and audit-safe rules.
